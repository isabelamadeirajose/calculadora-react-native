# Calculadora React Native

Aplicativo de calculadora simples desenvolvido com React Native + Expo.

## Sobre o projeto

Exercício da disciplina de **Soluções Mobile** — UNISATC  
Professor: Thyerri Mezzari

Objetivo: compreender o uso de componentes, frameworks visuais e Hooks em React Native.

## Conceitos aplicados

- `useState` para gerenciamento de estado
- Componentização com props (componente `Botao` reutilizável)
- React Native Paper como framework visual

## Tecnologias

- React Native
- Expo
- React Native Paper

## Como rodar

```bash
npm install
npx expo start
